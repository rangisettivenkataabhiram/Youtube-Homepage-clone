# Youtube-Homepage-clone
A responsive YouTube homepage clone built with pure HTML and CSS — featuring a fixed header with search bar, collapsible-style sidebar, and a dynamic video grid that adapts from 3 columns to 1 on smaller screens
YouTube Homepage Clone
A pixel-close recreation of the YouTube homepage built using only HTML5 and CSS3 — no JavaScript, no frameworks.
Features

Fixed top navigation bar with search input, voice search, notifications badge, and profile avatar
Fixed left sidebar with icon-based navigation tabs
Responsive video grid (3 → 2 → 1 columns) using CSS Grid and media queries
Video cards with thumbnail, duration overlay, channel profile picture, title, and stats
Hover effects on buttons, sidebar tabs, and interactive elements
Google Fonts (Roboto) integration

Tech Stack
HTML5 · CSS3 · CSS Grid · Flexbox · Responsive Design
